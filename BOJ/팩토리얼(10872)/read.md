# 문재

0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

출력
첫째 줄에 N!을 출력한다.


# 풀이

재귀 연습으로풀어본 문제.

N에서 재귀로 N을 감소하는 값을 계속 곱해준다.

만약 N이 1이되면, 재귀를 종료하고 1을 return한다.
```java
static int factorial(int N) {
		
		if (N == 1 || N == 0) {
			return 1;
		}
		
		return N * factorial(N - 1);
		
	}
```
